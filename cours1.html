🐍 Cours 1: Introduction à Python

🎯 Objectif

Comprendre ce qu’est Python, son origine et pourquoi il est un langage incontournable aujourd’hui.

📖 Histoire rapide

Python a été créé en 1991 par Guido van Rossum, un informaticien néerlandais. Son nom vient du spectacle “Monty Python”! Il a été conçu pour être lisible, simple, et très puissant. Aujourd’hui, il est utilisé par Google, Netflix, Meta, NASA et bien d’autres.

🔧 À quoi sert Python?

  • Développement web (Django, Flask)
  • Automatisation de tâches
  • Jeux vidéo, applications
  • Analyse de données / IA / Machine Learning
