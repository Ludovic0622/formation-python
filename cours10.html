📄 Cours 10: Les fichiers en Python

🎯 Objectif

Apprendre à ouvrir, lire, écrire et sauvegarder des données dans des fichiers texte avec Python.

📘 Explication

  • open(nom, mode) → ouvre un fichier
  • mode="r" → lecture
  • mode="w" → écriture (écrase le contenu existant)
  • mode="a" → ajout à la fin
  • with → permet de fermer automatiquement le fichier après utilisation

✍ Exemple de code

# Écriture dans un fichier
with open("notes.txt", "w") as f:
    f.write("Math: 16\nFrançais: 14")

# Lecture du fichier
with open("notes.txt", "r") as f:
    contenu = f.read()
    print("📑 Contenu du fichier:")
    print(contenu)
