⚠ Cours 11: Exceptions

🎯 Objectif

Apprendre à éviter les crashs et à gérer les erreurs avec les blocs try et except.

📘 Explication

  • try: on place le code susceptible de provoquer une erreur
  • except: on gère l’erreur si elle se produit (ex: division par zéro)
  • finally (optionnel): exécuté toujours, qu’il y ait eu une erreur ou non

✍ Exemple de code

# Exemple: division sécurisée
try:
    x = int(input("Nombre à diviser: "))
    y = int(input("Diviseur: "))
    resultat = x / y
    print("✅ Résultat:", resultat)
except ZeroDivisionError:
    print("❌ Erreur: division par zéro interdite")
except ValueError:
    print("❌ Erreur: tu dois entrer un nombre")
finally:
    print("📦 Fin de l’opération")
