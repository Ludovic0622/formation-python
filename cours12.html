🧠 Cours 12: Logique booléenne

🎯 Objectif

Maîtriser les opérateurs logiques (`and`, `or`, `not`) pour combiner des conditions et affiner les prises de décision dans le code.

📘 Explication

  • True et False sont les deux seules valeurs booléennes
  • and → toutes les conditions doivent être vraies
  • or → au moins une doit être vraie
  • not → inverse la valeur (ex: not True = False)

✍ Exemple de code

est_inscrit = True
age = 17

if est_inscrit and age>= 16:
    print("✅ Accès autorisé")
elif not est_inscrit:
    print("🔒 Tu dois t’inscrire d’abord")
else:
    print("🛑 Accès refusé: âge insuffisant")
