🎓 Projet mini 1: Carnet d’élève

🎯 Objectif

Créer un petit programme qui demande les notes d’un élève, les stocke dans une liste, calcule la moyenne, et affiche un message adapté.

📘 Recommandations

  • Utilise une liste pour stocker les notes
  • Fais une boucle pour saisir plusieurs notes
  • Utilise une fonction pour calculer la moyenne
  • Affiche un message conditionnel selon le résultat

✍ Exemple de structure

# Exemple: carnet de notes
notes = []
nb = int(input("Combien de matières? "))

for i in range(nb):
    note = float(input("Note " + str(i+1) + ": "))
    notes.append(note)

def moyenne(liste):
    return sum(liste) / len(liste)

moy = moyenne(notes)
print("📊 Moyenne:", round(moy, 2))

if moy>= 16:
    print("🏅 Excellent!")
elif moy>= 12:
    print("🙂 Moyenne correcte")
else:
    print("⚠ Besoin d’efforts")
