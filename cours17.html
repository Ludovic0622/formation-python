🏗 Cours 17: Classes & objets

🎯 Objectif

Créer des structures personnalisées appelées classes, pour regrouper des variables et fonctions dans un modèle, et instancier des objets.

📘 Explication

  • class → définit une classe (un modèle)
  • __init__() → fonction spéciale pour initialiser les attributs
  • Un objet est une instance de cette classe

✍ Exemple de code

# Définition de la classe
class Eleve:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age

    def presentation(self):
        print("Je m'appelle", self.nom, "et j'ai", self.age, "ans.")

# Création d’un objet
etu1 = Eleve("Lina", 16)
etu1.presentation()
