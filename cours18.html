🧰 Cours 18: Attributs & Méthodes

🎯 Objectif

Apprendre à ajouter, modifier et utiliser des données associées à un objet (attributs), et à lui faire exécuter des actions personnalisées (méthodes).

📘 Explication

  • Attribut = une information liée à un objet (ex: nom, score, niveau)
  • Méthode = une fonction définie dans une classe, utilisant self
  • Les objets peuvent appeler leurs propres méthodes avec objet.methode()

✍ Exemple de code

class Joueur:
    def __init__(self, pseudo):
        self.pseudo = pseudo
        self.vie = 100
        self.score = 0

    def attaquer(self):
        print(self.pseudo, "attaque l’ennemi ⚔")
        self.score += 10

    def etat(self):
        print("🧍", self.pseudo, "- Vie:", self.vie, "- Score:", self.score)

# Utilisation
j1 = Joueur("Alex")
j1.etat()
j1.attaquer()
j1.etat()
