🧬 Cours 19: Héritage

🎯 Objectif

Créer des classes spécialisées qui héritent des propriétés d’une classe “mère” et peuvent les modifier ou les étendre.

📘 Explication

  • Une classe “enfant” hérite d’une classe “parent” → class Enfant(Parent)
  • Elle récupère tous les attributs et méthodes de la classe parent
  • Elle peut redéfinir ou compléter les comportements existants

✍ Exemple de code

# Classe parent
class Utilisateur:
    def __init__(self, nom):
        self.nom = nom

    def saluer(self):
        print("👋 Bonjour,", self.nom)

# Classe enfant
class Admin(Utilisateur):
    def supprimer_utilisateur(self):
        print("⚠", self.nom, "a supprimé un utilisateur.")

# Utilisation
a1 = Admin("Alex")
a1.saluer()                 # méthode héritée
a1.supprimer_utilisateur()  # méthode propre
