📦 Cours 2: Variables & Types

🎯 Objectif

Savoir stocker des données dans des variables, et comprendre les principaux types de données en Python.

📘 Explication

Une variable permet de conserver une information temporairement.

  • int: entier (ex. 42)
  • float: nombre décimal (ex. 3.14)
  • str: texte (ex. "Hello")
  • bool: vrai ou faux (True ou False)

✍ Exemple de code

# Déclaration de variables
prenom = "Lina"
age = 25
moyenne = 14.5
est_inscrite = True

# Affichage
print("👩 Prénom:", prenom)
print("🎂 Âge:", age)
print("📊 Moyenne:", moyenne)
print("✅ Inscrite:", est_inscrite)
