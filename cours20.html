🎮 Projet mini 2: Jeu de combat

🎯 Objectif

Créer un petit jeu en ligne de commande où deux personnages s'affrontent. Chaque personnage a des attributs, peut attaquer et recevoir des dégâts.

📘 Recommandations

  • Crée une classe Personnage avec nom, vie, force
  • Crée une classe Guerrier ou Magicien qui hérite
  • Ajoute des méthodes: attaquer(), recevoir_degats(), etat()
  • Fais un combat entre deux objets dans une boucle

✍ Exemple de structure

import random

class Personnage:
    def _init_(self, nom, vie, force):
        self.nom = nom
        self.vie = vie
        self.force = force

    def attaquer(self, cible):
        degats = random.randint(1, self.force)
        print(f"{self.nom} attaque {cible.nom} 🔥 ({degats} dégats)")
        cible.recevoir_degats(degats)

    def recevoir_degats(self, valeur):
        self.vie -= valeur
        print(f"{self.nom} n’a plus que {self.vie} points de vie.")

    def est_vivant(self):
        return self.vie> 0

# Héritage
class Guerrier(Personnage):
    def crier(self):
        print(f"{self.nom} crie: Pour la gloire !")

# Combat
p1 = Guerrier("Thor", 100, 20)
p2 = Personnage("Gobelin", 60, 10)

while p1.est_vivant() and p2.est_vivant():
    p1.attaquer(p2)
    if p2.est_vivant():
        p2.attaquer(p1)
