📦 Cours 21: Bibliothèques utiles

🎯 Objectif

Découvrir des bibliothèques Python populaires pour enrichir tes programmes sans tout recréer toi-même.

📘 Bibliothèques standards

  • random → tirages aléatoires (dés, loteries, etc.)
  • math → fonctions mathématiques (racines, puissances, etc.)
  • datetime → gestion des dates et heures
  • os → manipuler les fichiers et dossiers

📦 Bibliothèques à installer

  • pip install numpy → calculs scientifiques, tableaux
  • pip install pandas → traitement de données tabulaires
  • pip install requests → accès à des sites web ou APIs
  • pip install matplotlib → créer des graphiques visuels

✍ Exemple de code

import random
print("🎲 Dé:", random.randint(1, 6))

import math
print("🧮 Racine de 49:", math.sqrt(49))

from datetime import datetime
print("🕰 Heure actuelle:", datetime.now())
