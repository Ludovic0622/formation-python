📚 Révision générale de la formation Python

🧠 À bien retenir

  • Variables: stockent des données (nombre, texte, etc.)
  • Types: int, float, str, bool, list, dict
  • Conditions: if, elif, else pour tester des cas
  • Boucles: for, while, et boucles imbriquées
  • Fonctions: créer avec def, renvoient des résultats avec return
  • Entrées/sorties: input(), print(), manipulation de fichiers
  • Listes & dictionnaires: structures de données souples
  • Exceptions: try / except pour gérer les erreurs
  • POO: classes, objets, méthodes, héritage
  • Modules: import, bibliothèques utiles (random, math, turtle…)

✨ Syntaxes fréquentes

# Déclaration de fonction
def carre(x):
    return x * x

# Boucle for
for i in range(5):
    print("🔁", i)

# Condition
if age>= 18:
    print("Majeur")
else:
    print("Mineur")
