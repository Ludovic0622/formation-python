🔀 Cours 3: Conditions

🎯 Objectif

Utiliser les conditions pour exécuter un code différent selon les situations (logique `si... alors`).

📘 Explication

  • if: si la condition est vraie, on exécute le bloc
  • elif: sinon si une autre condition est vraie
  • else: sinon, par défaut

On utilise les opérateurs de comparaison: ==, !=, <, >, etc.

✍ Exemple de code

age = int(input("Quel âge as-tu? "))

if age < 13:
    print("🚫 Accès interdit aux enfants.")
elif age < 18:
    print("🔐 Accès avec autorisation.")
else:
    print("✅ Bienvenue!")
