🔁 Cours 4: Les boucles

🎯 Objectif

Apprendre à répéter des actions automatiquement grâce aux boucles while et for.

📘 Explication

  • While → répète tant qu’une condition est vraie
  • For → répète un nombre défini de fois (ex: sur une plage range())

✍ Exemple de code

# Boucle while: compteur
i = 1
while i <= 5:
    print("💬 While:", i)
    i += 1

# Boucle for: affichage
for n in range(1, 6):
    print("🔢 For:", n)
