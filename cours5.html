🧩 Cours 5: Les fonctions

🎯 Objectif

Créer et utiliser des fonctions pour organiser son code, avec ou sans paramètres, et retourner des valeurs.

📘 Explication

Une fonction s’écrit avec def. Elle peut prendre des paramètres, exécuter du code et retourner un résultat avec return.

✍ Exemple de code

# Déclaration d’une fonction
def saluer(prenom):
    message = "Bonjour " + prenom
    return message

# Utilisation
resultat = saluer("Lina")
print(resultat)
