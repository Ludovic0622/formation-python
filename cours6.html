📚 Cours 6: Listes

🎯 Objectif

Apprendre à créer, modifier et parcourir des listes en Python, qui permettent de stocker plusieurs données dans un seul conteneur.

📘 Explication

  • Une liste est une séquence d’éléments entre crochets []
  • Chaque élément a un index (position), qui commence à 0
  • On peut accéder, ajouter, supprimer, trier, compter, etc.

✍ Exemple de code

# Création d’une liste
fruits = ["pomme", "banane", "kiwi"]

# Ajout d’un élément
fruits.append("fraise")

# Accès à un élément
print("Premier fruit:", fruits[0])

# Parcours de la liste
for fruit in fruits:
    print("🍎", fruit)
