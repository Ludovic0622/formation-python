💬 Cours 8: Entrées utilisateur

🎯 Objectif

Savoir récupérer une information que l’utilisateur tape au clavier avec la fonction input().

📘 Explication

  • input() affiche un message et attend que l’utilisateur écrive
  • Ce que l’utilisateur écrit est une chaîne de caractères (type str)
  • On peut le convertir en nombre avec int() ou float()

✍ Exemple de code

# Demander une info
nom = input("Quel est ton prénom? ")
print("👋 Bonjour", nom)

# Conversion en nombre
age = int(input("Quel âge as-tu? "))
print("🎂 Tu auras bientôt", age + 1, "ans")
