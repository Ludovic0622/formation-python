📁 Cours 9: Modules en Python

🎯 Objectif

Apprendre à importer et utiliser des modules Python pour étendre les fonctionnalités sans tout réécrire soi-même.

📘 Explication

  • import permet de charger un module
  • from... import... permet de charger une fonction précise
  • Modules standards: math, random, datetime, etc.

✍ Exemple de code

# Module math
import math

r = 4
aire = math.pi * r ** 2
print("🟢 Aire du cercle:", round(aire, 2))

# Module random
from random import randint
nb = randint(1, 100)
print("🎲 Nombre aléatoire:", nb)
