🎲 Projet 1: Devine le nombre

📋 Objectif

Créer un jeu dans lequel l’utilisateur doit deviner un nombre choisi aléatoirement par l’ordinateur, entre 1 et 100.

🧠 Notions utiles

  • Variables
  • Boucles while
  • Conditions if / else
  • Fonction random.randint()
  • Entrée utilisateur input()

📌 Exemple de code

import random

nombre_secret = random.randint(1, 100)
tentatives = 0
devine = 0

print("🧠 Devine le nombre entre 1 et 100")

while devine!= nombre_secret:
    devine = int(input("Ta proposition: "))
    tentatives += 1

    if devine < nombre_secret:
        print("📉 Trop petit.")
    elif devine> nombre_secret:
        print("📈 Trop grand.")
    else:
        print(f"🎉 Bravo! Tu as trouvé en {tentatives} tentatives.")
