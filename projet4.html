📈 Projet 4: Analyse d’un fichier CSV

📋 Objectif

Lire un fichier CSV contenant des données (ex: notes, prix) et calculer la moyenne, le maximum et le minimum.

🧠 Notions utiles

  • Manipulation de fichiers CSV
  • Bibliothèque csv
  • Boucles for
  • Calculs statistiques

📌 Exemple de code

import csv

valeurs = []

with open("data.csv", newline="") as fichier:
    lecteur = csv.reader(fichier)
    next(lecteur)  # Ignore l'en-tête
    for ligne in lecteur:
        valeur = float(ligne[1])  # Suppose que la valeur est en 2e colonne
        valeurs.append(valeur)

print("📊 Statistiques:")
print(f"Nombre d'entrées: {len(valeurs)}")
print(f"Moyenne: {sum(valeurs)/len(valeurs):.2f}")
print(f"Max: {max(valeurs)} – Min: {min(valeurs)}")
