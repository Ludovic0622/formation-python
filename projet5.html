🕹 Projet 5: Pierre – Papier – Ciseaux

📋 Objectif

Coder un jeu où l’utilisateur affronte l’ordinateur au célèbre jeu “pierre, papier, ciseaux” en Python.

🧠 Notions utiles

  • Fonction random.choice()
  • Conditions avec if / elif / else
  • Chaînes de caractères & entrée utilisateur

📌 Exemple de code

import random

options = ["pierre", "papier", "ciseaux"]
ordi = random.choice(options)
joueur = input("Ton choix (pierre, papier ou ciseaux): ").lower()

print(f"L’ordinateur a choisi: {ordi}")

if joueur == ordi:
    print("🤝 Égalité!")
elif (joueur == "pierre" and ordi == "ciseaux") or \
     (joueur == "papier" and ordi == "pierre") or \
     (joueur == "ciseaux" and ordi == "papier"):
    print("🎉 Tu gagnes!")
elif joueur in options:
    print("💻 L’ordinateur gagne.")
else:
    print("❌ Choix invalide!")
